# SmartWeather Dashboard

## Description
SmartWeather Dashboard is a responsive front-end weather application that allows users to search for a city and instantly view current weather conditions, coordinates, and a 5-day forecast. The application stores recent searches using localStorage and provides a clean and user-friendly interface built with Bootstrap.

## User Story
As a user, I want to search for a city and view current weather conditions and a multi-day forecast so that I can plan my day, commute, or travel more effectively.

## Acceptance Criteria
- Search for a city and display weather information
- Show current weather conditions
- Show a 5-day forecast
- Save recent city searches using localStorage
- Allow users to click a saved city to reload weather data
- Use Bootstrap for styling
- Use a modal instead of alerts, prompts, or confirms
- Be responsive across devices
